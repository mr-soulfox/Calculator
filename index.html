Calculadora padrão

é uma calculadora com as funçoes padrões

Os erros aparecerão aqui

Teclas permitidas:

  • C - Serve para apagar tudo
  • P - Serve para porcentagem
  • F - Serve para fração
  • E - serve para expoenciação
  • Q - serve para raiz quadrada
  • N - Serve para o simbolo -
  • ENTER - Mostrar o resultado
  • BACKSPACE - Apagar
